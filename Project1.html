Digital State Machine – Designed a Training Device

The Digital State Machine project involved the development and design of a training device that aimed to assess the button-pushing abilities of a test subject. The machine was implemented with two inputs and featured a total of 12 output signals, which included 10 yellow LEDs, 1 red LED, and 1 green LED. The circuit behavior was programmed to initiate in an initial state, where none of the LEDs were illuminated, and would remain in this state until an input was activated. To simulate the progression of the test subject's performance, the system was designed to advance to the next state and illuminate an additional yellow progress LED each time the correct input was activated. It would stay in this state until another input was activated. To account for incorrect inputs or simultaneous activation of two inputs, the system was programmed to remain in the same state if no input was detected. However, if an incorrect input or two inputs were activated simultaneously, the red LED would be activated for a single clock cycle. Finally, the system was designed to activate the green LED for one cycle upon successful completion of the correct sequence, indicating that the test subject had successfully achieved the desired outcome. Overall, the Digital State Machine project involved the implementation of a training device that effectively assessed a test subject's button-pushing abilities through the utilization of a Moore machine design and a carefully programmed system.
